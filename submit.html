Submission Guidelines & Link

How to Submit

IJCAI currently accepts manuscript submissions via the following temporary routes until ScholarOne/OJS is configured:

  • EasyChair (recommended temporary): Create an EasyChair event and use its submission link here. (Placeholder)
    https://easychair.org/conferences/?conf=ijcaijournal2025
  • Email (backup): editor@ijcai-journal.org (attach manuscript in DOCX + PDF)

Required Files

  • Main manuscript (DOCX)
  • Anonymous PDF (for review)
  • Cover letter (optional)
  • Author ORCID IDs
  • Ethical approval statement (if applicable)

Review Timeline

Initial editorial screening: 3–7 days. Peer review: 2–4 weeks. Final decision typically within 6 weeks.

Note: When ScholarOne is set up you will be notified and the official submission link will replace this page.
